# Kubeservice Lxcfs WebHook

`LXCFS` is a small FUSE filesystem written with the intention of making Linux containers feel more like a virtual machine.

[Kubeservice Lxcfs WebHook](https://github.com/kubeservice-stack/lxcfs-webhook) is a Kubernetes Admission Webhook for Linux Container Resources View.

This chart creates a [Kubeservice Lxcfs WebHook](https://github.com/kubeservice-stack/lxcfs-webhook) deployment on a [Kubernetes](http://kubernetes.io) cluster using the [Helm](https://helm.sh) package manager.
